Assignment

For my final React course assignment I was asked to build a fullstack CRUD web API application with ASP .NET Core and React.js.

Result

I first created a MS SQL database with Northwind example data which I then connected to a RESTful web API built with ASP .NET Core. I also built a user authorization service for the web API. I published both the API and the database to a Azure cloud hosted server. After that I built the frontend side of the project using React.js which I installed to Node.js along with several other Node.js packages such as Axios for HTTP requests, md5 for password hashing and React-Bootstrap, React-Icons and SASS for styling purposes.

Assignment
I wanted to build a language learning memory game, where you match Japanese number kanji cards with their hiragana counterparts.

Result

The app is a simple HTML/CSS/Vanilla JavaScript memory game where you first click a Start Game button to make the cards appear. You can then reset the game by clicking on the Start Game button which now has the Reset Game text on it. The game keeps track on your score and the mistakes you've made.
